# INTELLEGANT-TRASH-DISPOSAL-MACHINE


The Intelligent Trash Disposal Machine is an automated, eco-friendly  system aimed at streamlining the recycling process. Powered by a well-trained AI model, the system recognizes and specifies waste items such as plastic, paper, and others. Using a camera, sensors, and servo motors, it then automatically opens the appropriate bin lid.  Awarded 2nd Place in a Hackathon!

# Key Feature & Impact
Automated Sorting: Automatically identifies the proper lid to open based on the detected item.

# Tech Stack
Hardware: ESP32, Ultrasonic Sensors, Servo Motors.

Software: Python, YOLOv11 (Computer Vision Object Detection), C++/Arduino (Microcontroller Logic and Serial Communication).
